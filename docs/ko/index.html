한 번 작성, 어디서나 배포

하나의 .md 규칙 작성. Cursor용 .mdc와 Codex용 AGENTS.md로 자동 변환. 도구 간 복사-붙여넣기 불필요.

#

팀 전체 일관성

Git으로 규칙 공유. 모든 팀원이 동일한 소스에서 설치. update 명령 한 번으로 전원 동기화.

$

로컬 수정 보호

설치 및 업데이트 시 로컬 커스터마이징을 절대 덮어쓰지 않음. 새 파일만 추가. 당신의 규칙, 당신의 통제.

~

설정 없이 바로 시작

명령어 하나로 설치. 대화형 위저드가 도구 선택, 규칙 카테고리, 템플릿을 안내. 키워드 폴백으로 즉시 사용 가능.

@

커뮤니티 마켓플레이스

로컬 웹 UI에서 커뮤니티 규칙을 검색, 설치, 제거. PR로 기여하면 머지 후 즉시 반영.

명령어

모든 워크플로를 위한 간단한 CLI

ai-nexus install 규칙을 전역으로 설치 (~/.claude/)
ai-nexus install -i 대화형 모드 - 도구 및 규칙 선택
ai-nexus init 현재 프로젝트에 설치
ai-nexus update 최신 규칙 동기화 (로컬 변경 유지)
ai-nexus doctor 설치 문제 진단
ai-nexus browse 브라우저에서 규칙 마켓플레이스 열기
ai-nexus search [keyword] 레지스트리에서 커뮤니티 규칙 검색
ai-nexus test <prompt> 어떤 규칙이 로딩될지 미리보기

규칙 마켓플레이스

브라우저에서 규칙을 탐색, 설치, 관리

$ npx ai-nexus browse
